# Source Code Directory

This is the main source code directory for the Worker Node NestJS application.

## Main Files

- **main.ts**: The entry point of the application that bootstraps the NestJS module.
- **app.module.ts**: The root module that imports and configures all other modules.
- **app.controller.ts**: The root controller for basic routes.
- **app.service.ts**: The root service for basic application logic.

## Directory Structure

- **controllers/**: HTTP request handlers
- **services/**: Business logic implementation
- **repositories/**: Data access operations
- **entities/**: Domain data models
- **dtos/**: Data Transfer Objects
- **config/**: Application configuration
- **middleware/**: HTTP middleware
- **utils/**: Utility functions
- **interfaces/**: TypeScript interfaces
- **infrastructure/**: External services integration
- **modules/**: Feature modules
- **health/**: Health check endpoints
